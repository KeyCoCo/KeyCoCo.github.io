Java基础语法

Java基本数据类型（值类型）

在Java中共有4大基本数据类型分别为：整数型，小数型，字符型，boolean型

整数型：byte,short,int,long

小数型：float,double

字符型：char

boolean型：true,false

基本数据类型之间的转换
自动转换 注：Java在底层运算时会自动将byte,short,char类型转换为int类型进行计算
graph LR
   byte --> short
   short --> int
   char --> int
   int --> long
   float --> double

int a=1+'A';//此时字符类型自动转换为int类型
System.out.println(a);//结果为66
int i='A'+'B';
System.out.println(i);//结果为131
强制转换 注：在强制转换时可能会造成数据溢出
graph LR
   double --> float
   long --> int
   int --> short
   short --> byte
   int --> char
   short --> char
   byte --> char

long l=987654321987654L;
int i=(int)l;
System.out.println(i);//结果为822468678内存溢出
char c=(char)65;
System.out.println(c);//结果为A

Java引用数据类型

在Java语言中除了基本数据类型以外，其他类型的数据都是引用数据类型（包括对象class）。在Java中为所有的基本数据类型提供了所对应的包装类型。除了Integer与Character外其他包装类皆是将首字母大写即可。包装类型与基本类型之间的转换成为拆装箱。

int i=34;
Integer ie=new Integer(i);//自动装箱
double d=3.14;
Double db=new Double(d);//自动装箱
int k=ie;//自动拆箱

String是一个特殊的引用类型（字符串类型），它的底层由字符数组构成，他的内容不可被修改，只能被重新赋值。

Java也提供了相应的可变字符串类型StringBuffer与StringBuilder它们都继承了一个抽象的字符串父类AbstractStringBuilder。StringBuffer与StringBuilder的区别。StringBuffer的所有公开方法都被修饰符synchronized同步所有它的线程是安全的但是性能较低。而StringBuilder并没有被synchronized修饰所有它的线程是不安全的但极大的提高了性能。

StringBuffer sb=new StringBuffer("I Love You!");
sb.append(" java");//在指定字符串后面添加字符串
System.out.println(sb);//结果:I Love You! java

基本数据类型与引用数据类型的区别

基本数据类型储存在栈里，引用数据类型储存在堆里

public class TestClass1{
    private String Name;
    private int Age,Height;
    public TestClass(String Name,int Age,int Height){
        this.Name=Name;
        this.Age=Age;
        this.Height=Height;
    }
}

public class TestClass2{
    private String Name;
    private int Age,Height;
    public TestClass(String Name,int Age,int Height){
        this.Name=Name;
        this.Age=Age;
        this.Height=Height;
    }
}

public class Test{
    public static void main(String [] args){
        int i = 1;
        float f = 3.14;
        String str1 = new String("Hello word!");
        TestClass1 class1=new TestClass1("小明",18,1.74);
        TestClass2 class2=new TestClass2("小红",17,1.64);
    }
}
