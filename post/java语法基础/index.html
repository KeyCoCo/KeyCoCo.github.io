Java基本数据类型（值类型）

在Java中共有4大基本数据类型分别为：整数型，小数型，字符型，boolean型

整数型：byte,short,int,long

小数型：float,double

字符型：char

boolean型：true,false

基本数据类型之间的转换
自动转换 注：Java在底层运算时会自动将byte,short,char类型转换为int类型进行计算
graph LR
   byte --> short
   short --> int
   char --> int
   int --> long
   float --> double
   int --> float

int a=1+'A';//此时字符类型自动转换为int类型
System.out.println(a);//结果为66
int i='A'+'B';

Java引用数据类型

在Java语言中除了基本数据类型以外，其他类型的数据都是引用数据类型（包括对象class）。在Java中为所有的基本数据类型提供了所对应的包装类型。除了Integer与Character外其他包装类皆是将首字母大写即可。包装类型与基本类型之间的转换成为拆装箱。

int i=34;
Integer ie=new Integer(i);//自动装箱
double d=3.14;
Double db=new Double(d);//自动装箱
int k=ie;//自动拆箱

String是一个特殊的引用类型（字符串类型），它的底层由字符数组构成，他的内容不可被修改，只能被重新赋值。

Java也提供了相应的可变字符串类型StringBuffer与StringBuilder它们都继承了一个抽象的字符串父类AbstractStringBuilder。StringBuffer与StringBuilder的区别。StringBuffer的所有公开方法都被修饰符synchronized同步所有它的线程是安全的但是性能较低。而StringBuilder并没有被synchronized修饰所有它的线程是不安全的但极大的提高了性能。

StringBuffer sb=new StringBuffer("I Love You!");
sb.append(" java");//在指定字符串后面添加字符串
System.out.println(sb);//结果:I Love You! java

Java中的运算符

在Java中有5种运算符

基本运算符：+ - * / %

比较运算符：> < >= <= !=

逻辑运算符：&&（短路与）一假即为假，||（短路或）一真即为真，!（非）真即假 假即真，&（与）真真即为真，|（或）假假即为假

三目运算符：String str = (true&&false)“false”:“true”;

位运算符：» « ~ ^ & |

Java中的程序运行结构

在Java中有三种运行结构

顺序结构：程序从上到下依次执行。

循环结构：在某个代码块中重复执行相同的操作。

分支结构：在特定的条件下执行设置好的代码块。

循环结构
while(true){
    System.out.println("I Love China!");
}

do{
    System.out.println("I Love China!");
}while(true);

for(int i=0;i<1000;i++){
    System.out.println("Hello World!");
}

int [] intList={1,2,3,4,5};
for(int i:intList){
    System.out.println(i);
}
分支结构
int age = 12;
if(age < 15){
    System.out.println("儿童票");
}

int achievement = 78;
if(achievement >= 90){
    System.out.println("A");
}else if(achievement >= 70){
    System.out.println("B");
}else if(achievement >= 50){
    System.out.println("C");
}else{
    System.out.println("D");
}

char size = 'L';
switch(size){
        case 'XL':
        System.out.println("特大杯");
        case 'L':
        System.out.println("大杯");

        System.out.println("中杯");

        System.out.println("小杯");
    	default:

Java中的修饰符与关键字

在Java中为了确保程序的安全与可维护型添加了修饰符。关键字是计算机事先定义好的，有特别含义的标识符。注：在Java中标识符不能用修饰符或关键字命名。

访问修饰符：

public（公共的） 修饰变量，类，方法，构造方法

private（私有的）修饰 变量，方法

protected（受保护的）修饰变量，方法

默认的 修饰变量，类，方法，构造方法

非访问修饰符：

static（静态的）修饰变量，方法

final（不可改变的）修饰变量

abstract（抽象的）修饰抽象类，抽象方法

synchronized（同步的）用来同步线程

transient（瞬变的）使序列化的对象不会持久化

volatile（不稳定的）

常用关键字：

break continue void extends implements interface
