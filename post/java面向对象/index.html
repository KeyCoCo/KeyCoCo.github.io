Java面向对象

面向对象概述

在Java中一切皆为对象，面向对象是对类的描述。面向对象更加趋近于自然语言，在Java中面向对象含有属性和方法，我们通过操作面向对象中的属性和方法来满足相应的需求。下图的学生类包括了属性姓名、年龄、性别，和方法上学、放学、写作业。注意：Student是一个构造方法它用来初始化对象的属性。

我们在操作对象（类）调用方法时，通常需要将对象实例化当然我们也可以通过实例来访问对象的属性。注意：当变量或方法不被修饰符private修饰时。

public class MainTest(){
    public static void main(String[] args){
        Student student = new Student("小明",19,'男');
        student.goSchool();
        student.goHome();
        student.doHomework();
        System.out.println("姓名：%s%t年龄：%s%t性别：%s%n",student.Name,student.age,student.sex);
    }
}

变量作用域

在类内方法体外定义的变量为成员变量，又称实例变量。成员变量被对象名访问。

当成员变量被static修饰时进化为类变量，又称静态变量。类变量被类名访问。

在方法体中定义的变量为局部变量它只能在该方法体中访问。

面向对象的三大特性

封装

封装用private修饰成员变量，提供相应的get和set方法来获取变量提高里程序的安全性。

继承

通过继承父类使得子类拥有父类相应的属性与方法，Java中的继承为单继承即一个父类能被多个子类所继承，但一个子类只能继承一个父类，提高了代码的可拓展性。注意：子类不能继承父类私有的方法与属性。

关键字：子类 extends 父类 （this表示本类 super表示父类 解决子类与父类变量名或方法名重名。）

多态

通过定义不同的接口来实现相应的功能，大大提高了代码的可维护性与可持续开发性。

关键字：interface 接口名 子类名 implements 接口名

伪代码：@Override重写（方法名参数列表返回值一致）@Overload重载（方法名一致参数列表返回值随机不同）
